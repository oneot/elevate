## 2.  Microsoft Learn 가입

- **i) MS Learn이란?**
    
    MS Learn(learn.microsoft.com)은 **마이크로소프트가 운영하는 공식 학습 플랫폼**으로,
    개발자·IT 전문가·교육자·학생 등 누구나 **무료로 ms플랫폼과 도구를 배우고 실습하는** 공간입니다.
    
- **ii) MS Learn 가입**
    1.[Microsoft Learn](https://learn.microsoft.com/ko-kr/) 를 클릭하여 Microsoft Learn으로 이동합니다.

    2.우측 상단의 로그인 버튼을 클릭한 뒤 MEE 계정으로 로그인 합니다.
    
    3.로그인 하고 프로필을 클릭하면 이런 화면을 볼 수 있게 됩니다.
    
    ![image.png](/images/mee/explorer-procedure/image9.png)
    
    4.프로필 화면에서는 나의 Achievments를 확인할 수 있습니다. 
    Achievements → Learning Paths 를 클릭하면, 완료한 Learning Paths 확인 가능
    
    ![image.png](/images/mee/explorer-procedure/image10.png)
    
    MEE Explorer가 되기 위해서는 아래 두 트로피가 Achievements → Learning Paths에 존재해야 합니다. 
    
    Learning path를 이수하는 방법은 아래에서 계속됩니다.
    
    ![IMG_9032.jpeg](/images/mee/explorer-procedure/image11.jpeg)
    
    ![IMG_9033.jpeg](/images/mee/explorer-procedure/image12.jpeg)
    
- **iii) MS Learn Username설정(MEE explorer 최종 지원을 위해 중요한 부분**
    
    **Username**을 정확히 알고 있어야 MEE explorer 최종 지원이 가능하고, **Username**가 잘못된 경우 **반려**됩니다.(Learn에서의 이수내역을 확인하기 위함.)

    2.USERNAME을 설정합니다. 
    
    > USERNAME은 기존 숫자 그대로 두셔도 되고 원하시는 이름으로 설정하셔도 됩니다.
    
    
    3.USERNAME은 따로 메모해두시거나 이후에 까먹으면 아래 사진의 방법대로 확인해주시면 됩니다.

## 3.  MEE  지원을 위해 필요한 학습 경로를 완료하기

- **Learning Path(학습 경로)란?**
    
    Learning path는 여러 개의 Module로 구성되어 있습니다. 
    
    - **학습 경로(Learning Path)** 단위로 이수 여부를 확인합니다.
    - 실제 진행은 **경로 속의 모듈을 하나씩 차근차근 완료**하는 방식입니다.
    
    
    > 가장 이상적인 방법은 모듈 안의 모든 유닛을 꼼꼼히 살펴보고 학습하는 것이지만, </br>
    이 자료에서는 **‘이수**’ 및 **‘선발’**에 관점을 두고 작성해보았으니 참고 부탁드립니다. </br>
    저의 경우는 지원 기간의 막바지에 알게 되어 시간이 모자랐고 우선 모두 이수했습니다. 완료한 **Learningpath**라고 해도 언제든 다시 학습할 수 있습니다. </br>
    이후 **Teams**나 **Copilot**에 대해 공부하고 싶은 경우 
    [**MS Learn사이트**](https://learn.microsoft.com/ko-kr/)에서 관심 분야를 검색해 공부했습니다. </br>
    공식 교육 자료인만큼 퀄리티가 아주 훌륭합니다. ^^
