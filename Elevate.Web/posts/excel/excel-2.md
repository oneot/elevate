# Excel로 급식지도 담당 배정하기

> **이런 순간, 익숙하지 않으신가요?** <br>
새 학기가 되고, 학생들이 학교에 오면서 급식지도 당번을 정해야 합니다. 
그러나 학급 급식 당번을 배정할 때마다 매번 일정을 맞추고, 중복되지 않도록 관리하는 과정은 생각보다 많은 시간이 듭니다. 특히 업무가 몰리는 시기에는 이 작업만으로도 업무 부담이 커지곤 합니다. Excel, 그리고 Copilot을 함께 사용하면 선생님이 기본적인 정보만 입력해도 자동으로 급식 당번을 적절하게 배정할 수 있어, 반복 작업에 쓰던 시간을 크게 줄일 수 있습니다.

# STEP 2. 프롬프트로 교사 데이터 시트 만들기

정확한 배정을 위해 선생님들의 정보 표가 필요합니다. Excel 내의 Copilot에 아래 프롬프트를 입력합니다.

- 에이전트 모드를 사용해야 문서 내부 편집이 원활하게 진행될 수 있습니다.
- 🔄 파일을 반드시 Onedrive에 올려야 에이전트 모드를 사용할 수 있습니다.

```
새 시트 ‘교사정보’를 만들고 아래 표를 생성해줘. Teacher | MaxDuty | 불가능요일 (여기는 내가 수동으로 채울게)
```

# STEP 4. Copilot 활용하기

기존 **3월 급식당번** 시트(또는 동일 시트)에서 월~금 격자 중 ‘날짜가 들어있는 칸 바로 아래 줄’을 **배정 입력 칸**으로 쓸 거예요.

- 예: 날짜(2,3,4,5,6)가 있는 줄 바로 아래 줄을 “배정 교사” 줄로 사용하고, 이후 각 주(1주~5주)도 동일 패턴
- 아래 프롬프트를 Excel의 Copilot에 그대로 붙여넣으세요. (달력 시트 이름과 입력 범위는 파일에 맞게 살짝 고쳐 부르기만 하면 됩니다.)

```
다음 요구사항을 정확히 수행해줘.

[목표]
- '교사정보' 시트의 표(헤더: Teacher, MaxDuty, 불가능요일)를 유일한 기준으로 삼아,
- '3월 급식당번' 시트의 달력 격자에 배정 교사 이름을 자동으로 채워 넣어.
- 경고/조건부서식/요약표는 만들지 말고, 결과만 채워.
- 각 교사는 자신의 MaxDuty를 넘기지 않도록 하고,
- 각 교사의 '불가능요일'(월/화/수/목/금)은 절대 배정하지 마.
- 달력은 평일(월~금)만 고려하고, 비어 있는 날짜 셀은 건너뛰어.

[입력 데이터]
- 시트: '교사정보'  
  - 열 A: Teacher(텍스트)  
  - 열 B: MaxDuty(정수)  
  - 열 C: 불가능요일(예: '월,수' 처럼 콤마로 구분된 텍스트)

[달력 구조]
- 시트: '3월 급식당번'
- 달력은 월~금 5열 구조이며, 각 주(1주~5주) 행에 '날짜 숫자'가 있고, 그 바로 아래 행이 '배정 교사'를 적는 입력 칸이야.
- 이미 적혀 있는 날짜 숫자는 그대로 두고, 그 아래 빈칸들(배정 칸)에만 교사 이름을 넣어.

[배정 규칙]
1) MaxDuty를 넘기지 않는다.
2) 불가능요일에는 배정하지 않는다.
3) 가능한 한 교사별로 공평하게(라운드로빈에 가깝게) 분배한다.
4) 날짜가 비어 있거나(해당 요일에 날짜가 없거나) 주말은 건너뛴다.
5) 동일 주 안에서도 위 규칙을 지키되, 전체 월 단위 균형을 우선으로 한다.

[실행 방법]
- '교사정보' 표를 내부 목록으로 읽어오고, 각 교사별로 '남은 가능 횟수' 카운터를 만든 뒤,
- 달력의 각 평일 칸을 순서대로 순회하면서, 해당 요일에 배정 가능한 교사들 중 '남은 가능 횟수'가 있는 사람을 공평하게 배치해.
- 배치할 수 없는 칸은 비워둬(추가 지시는 하지 말 것).

[출력]
- '3월 급식당번' 시트의 배정 칸에 최종 교사 이름만 채워 넣어.
- 다른 시트나 서식은 만들지 말 것.
```
