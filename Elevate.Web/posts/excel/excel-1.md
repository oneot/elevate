> **이런 순간, 익숙하지 않으신가요?** <br>
시험 기간마다 과목별 담당 교사들이 보내는 성적 엑셀 파일을 모아 하나의 통합 자료로 만드는 일은 생각보다 많은 시간이 걸립니다. 매번 파일을 열어 형식을 맞추고, 피벗테이블을 새로 만들어 학급별·과목별·성별 성적을 다시 분석하다 보면 정작 결과 해석과 학생 피드백에 쓸 시간이 부족해집니다. Excel의 Copilot과 자동화된 정리·분석 흐름을 미리 만들어 두면, 폴더에 성적 파일만 모아 두어도 자동으로 통합되고 피벗테이블까지 즉시 생성되어 분석 결과가 바로 나타납니다. 교사는 반복 정리 작업 대신, 몇 초 만에 완성된 통계와 비교 자료를 바탕으로 개별 상담 준비와 성취도 향상 전략 수립에 더 많은 시간을 쓸 수 있습니다.

# STEP 1. 하나의 폴더에 성적 파일 모으기

우선 우리 반 학생들의 각 과목 성적 파일을 하나의 폴더에 모을거예요. 아래는 예시 성적 파일입니다.

-❗원활한 진행을 위해 onedrive에 파일을 넣어주세요

📋 2026-1학기 국어 성적 (1학기), 2026-1학기 수학 성적 (1학기), 2026-1학기 통합사회 성적 (1학기), 2026-1학기 통합과학 성적 (1학기), 2026-1학기 영어 성적 (1학기)

![image](/images/m365/excel-1/img0.png)

# STEP 2. 통합 문서 만들기

이제 성적을 통합하여 분석할 새로운 Excel 파일을 만들거예요. Onedrive 내에서 '성적 통합 분석'이라는 파일을 만들어 봅시다. (동일한 폴더 내에서 만드는 것이 아닙니다!)

![image](/images/m365/excel-1/img1.png)

# STEP 3. 파일 자동으로 불러오기

통합 파일 내에서 '데이터' → '데이터 가져오기(미리 보기)'

![image](/images/m365/excel-1/img2.png)

데이터 가져오기(미리 보기)에서 '폴더'를 누르고, 성적표를 넣어둔 폴더를 선택합니다.

- [결합]에서 데이터 결합 및 변환을 누르고 기다립니다.

![image](/images/m365/excel-1/img3.png)

![image](/images/m365/excel-1/img4.png)

![image](/images/m365/excel-1/img5.png)

통합하기 전에 열 이름 통일, 필요 없는 열 제거, 데이터 바꾸기 등 정제를 한 번 해야겠죠? 성적표를 누르고 [확인]을 눌러주세요. 성적표가 기준이 됩니다.

- 성적표를 더블클릭해도 됩니다!

![image](/images/m365/excel-1/img6.png)

미리보기가 정상적으로 확인된다면, 잘 진행되고 있다는 뜻입니다.

이제 [닫기 및 로드] 를 눌러 통합을 마무리합니다.

![image](/images/m365/excel-1/img6.png)

💡 성공적으로 통합되었다면 이런 화면을 볼 수 있어요!

- 이제 아래를 자동으로 관리됩니다:
    - 모든 파일 통합 	
    - 파일명이 달라도 통합 	
    - 열 순서/이름 달라도 통합 
    - 새 파일 넣으면 자동 통합

![image](/images/m365/excel-1/img7.png)

Copilot이 다음을 자동으로 수행해줄거예요.

- 파일별 서로 다른 열 이름 정규화 
- 학생 데이터 행 단위로 통합 
- 과목 코드/표기 방식 차이도 통일 
- 결측치·중복값 검토

# STEP 4. 분석을 위한 구조 자동 생성하기

통합된 표가 만들어지면 이제 알고 싶은 것을 Copilot에게 추가로 요청할건데요, 원하는 것을 골라 요청하면 됩니다.
아래는 과목별, 학생 개별을 주로 트래킹하는 예시 프롬프트예요. 분석해준 주요 지표를 한 눈에 볼 수 있도록 자동 시각화도 함께 생성해볼 겁니다.

```
통합된 성적 데이터를 기반으로 아래 두 가지 피벗테이블을 만들어줘.
1) 과목별 성적 요약 피벗테이블 - 행: 과목명(Source.Name 또는 과목명 열) - 값: 원점수(100), 수행평가(100), 중간고사(100), 기말고사(100)가 있다면 각각 평균 - 레이아웃은 표 형식 - “피벗 분석” 시트의 A1부터 배치
2) 학생 개별 성적 분석 피벗테이블 - 행: 학번 → 이름 - 값: 원점수(100), 수행평가(100), 중간/기말 점수 평균 - 필터: 과목명 - “피벗 분석” 시트 오른쪽 또는 아래쪽 빈 공간에 배치
두 피벗 모두 보기 좋게 자동 서식도 적용해줘.
```
![image](/images/m365/excel-1/img8.png)

# STEP 5. 분석 리포트 자동으로 작성하기

Copilot을 통해 피벗 분석 결과를 토대로 리포트 초안까지 생성할 수 있어요.

```
이번 시험 결과 요약 리포트를 작성해 주세요.
강점·보완점과 과목별 편차, 학생 상담에 필요한 인사이트를 포함해 주세요.
```

# STEP 6. 업데이트하기

이번에 만들어둔 대시보드는 다음에도 사용할 수 있어요!

- 피벗 테이블, 차트가 자동 업데이트되며
- 요약 리포트도 재생성됩니다
