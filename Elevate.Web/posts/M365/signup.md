“Microsoft 365 계정부터 만들라는데… 어디서 만들라는 거지?”  
공식 사이트에 들어갔더니 학교 계정이 아니라는 말이 나오고  
주변 선생님들은 다들 교육청 사이트 링크를 주는 상황  

👉 **결론부터 말하면,**  
국내 초·중·고 교사는 Microsoft 공식 사이트가 아니라  
**‘소속 교육청 사이트’에서 계정을 만드는 것이 맞습니다.**

---

# STEP 1. 어디서 만들어야 할까?

## 왜 공식 사이트가 아닌가요?

국내 초·중·고의 Microsoft 365는  
Microsoft가 직접 관리하는 개인 계정 구조가 아닙니다.

👉 시·도교육청이 하나의 조직으로 계약·운영하고,  
교사·학생 계정을 교육청 기준으로 통합 관리합니다.

그래서 아래와 같은 관리가 가능합니다.

- 학교 소속 확인  
- 교사 / 학생 계정 구분  
- 전근·전출·졸업 시 계정 정리  
- Teams 수업 권한 관리  

✅ 이 때문에 계정 생성도  
👉 **반드시 교육청 전용 M365 포털을 통해 진행해야 합니다.**

---

# STEP 2. 우리 학교 ‘교육청 M365 포털’ 찾기

각 시·도교육청마다 전용 계정 생성 사이트가 있습니다.

## 📌 예시 (일부)

- 서울특별시교육청 : https://o365.sen.go.kr  
- 경기도교육청 : https://www.goedu.kr  
- 부산광역시교육청 : https://o365.pen.go.kr  
- 강원특별자치도교육청 : https://office365.gwe.go.kr  
- 세종특별자치시교육청 : https://o365.sje.go.kr  

💡 보통 포털 이름은  
**“Microsoft 365 클라우드 서비스” / “Office 365 지원 시스템”** 으로 되어 있습니다.

---

# STEP 3. 교육청 사이트에서 계정 만들기

교육청마다 화면은 조금씩 다르지만,  
흐름은 거의 동일합니다.

## 1️⃣ 교육청 M365 포털 접속  
→ **[교직원 계정 생성]** 선택

## 2️⃣ 약관 동의 및 본인 인증  
- 휴대폰 인증  
- 또는 교육청 인증 절차 진행  

## 3️⃣ 학교 인증코드 입력  
- 학교 정보부 / 담당부서에서 관리  
- 학생은 가정통신문, 교사는 학교 내부 안내로 전달  

## 4️⃣ 계정 정보 입력  
- 아이디 / 비밀번호 설정  
- 보조 이메일 등록  

✅ 이 과정을 마치면  
교육청 도메인의 M365 계정이 생성됩니다.  
(예: @sen.go.kr, @학교명.go.kr 등)

---

# STEP 4. 계정 생성이 어렵다면?

계정 생성을 하다 막히는 부분이 있다면  
**microsoft-elevate.com** 에서 에이전트를 만나보세요.
![image](/images/m365/signup/img1.png)
1️⃣ microsoft-elevate.com 에 접속합니다.  
![image](/images/m365/signup/img2.png)
2️⃣ 우측 아래 **에이전트 버튼**을 클릭합니다.
![image](/images/m365/signup/img3.png)
3️⃣ 계정 생성과 관련된 부분을 질문합니다.

👉 microsoft-elevate 에이전트가 단계별로 안내해드립니다.

---

# STEP 5. 계정 생성 후 바로 할 수 있는 것들

생성된 계정으로 아래 사이트에 로그인해보세요.

- https://www.office.com  
- https://teams.microsoft.com  

로그인 즉시:

✅ Microsoft Teams 수업 팀 생성  
✅ 과제·퀴즈 게시  
✅ Word / Excel / PowerPoint 사용  
✅ OneDrive 자동 생성  

👉 수업용 M365 환경이 바로 활성화됩니다.

---

# 🎉 마무리

M365 교육용 계정은  
단순한 ‘오피스 계정’이 아니라,

✅ 학교 수업을 운영하기 위한 공식 계정입니다.

- Teams 수업  
- 과제·퀴즈  
- 학생 계정 연동  
- 학적 기반 관리  

이 모든 출발점이  
👉 **교육청 M365 계정입니다.**

---

💡 **TIP – 이렇게 이어서 활용하세요**

- Teams 수업 시작  
→ 교육청 계정으로 로그인하면 바로 수업 팀 생성 가능  

- 학생 계정 연동  
→ 같은 교육청 계정 체계라 초대·관리 간단  

- 학교 공식 기록과 연계  
→ 과제, 활동 이력이 자연스럽게 누적
