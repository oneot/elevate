# STEP 1. Onedrive

## Onedrive란?

- 모든 파일을 어디서나 안전하게 저장하고 공유해서 협업할 수 있게 해주는 클라우드 드라이브

## 교육용 계정으로 사용했을 때는…

- 전체 사용자에게 1TB의 무료 용량을 제공합니다. (학교에 따라 차이가 있을 수 있어요)
    - 용량 제한 없이 다양한 파일 작업을 할 수 있어요.
    - 안정적인 백업과 버전 관리가 가능합니다.
- Microsoft 교육 애플리케이션과 자연스럽게 연결됩니다.
    - Microsoft Teams for Education, OneNote Class Notebook 등 모든 교육 애플리케이션의 저장소 역할을 수행해요.
    - 수업자료 업로드나 과제 제출 파일, 그룹 프로젝트 파일 등을 자동으로 저장해요.
    - 단순한 문서 공유를 넘어서 교육 시나리오에 맞춘 ‘협업’ 지원이 가능해요.
- 학생들과 자료 공유를 훨씬 안정적이고 체계적으로 할 수 있습니다.
    - 학교 계정 내에서 파일이나 폴더 권한이 자동으로 설정됩니다
    - 반 전체, 특정 그룹, 특정 학생에게 안전하게 파일을 공유할 수 있어요.

# STEP 2. 웹에서 Onedrive 바로 써보기

1. 검색창에 ‘onedrive’을 입력하고, Microsoft 홈페이지에 들어갑니다.

![image](/images/m365/onedrive-tutorial/1.png)

2. 오른쪽 상단에서 로그인을 누른 후, 교육용 계정(교사/학생)으로 로그인을 진행합니다.

![image](/images/m365/onedrive-tutorial/2.png)

![image](/images/m365/onedrive-tutorial/3.png)

3. 이제 Onedrvie에서 작업을 시작할 수 있습니다!

![image](/images/m365/onedrive-tutorial/4.png)

- Tip! 최근에 Onedrive에서 작업한 파일을 ‘사용자 전용’이나 ‘최근’ 에서 확인할 수 있어요!

4. ‘만들기 또는 업로드’를 눌러서 바로 Onedrive 작업을 시작해보세요.

![image](/images/m365/onedrive-tutorial/5.png)

- Tip! 왼쪽 상단의 메뉴를 누를 경우 다른 Microsoft 365 앱들을 바로 사용할 수 있어요!

![image](/images/m365/onedrive-tutorial/6.png)

# STEP 3. 앱에서 고도화된 작업해보기

1. 아래의 링크로 들어가서 ‘앱 설치’ → ‘Microsoft 365 앱’을 선택합니다.

[앱 | M365 Copilot](https://m365.cloud.microsoft/apps?auth=2)

![image](/images/m365/onedrive-tutorial/7.png)

2. ‘Office 설치’를 누른 후 다운받아진 ‘OfficeSetup.exe’을 누릅니다.

![image](/images/m365/onedrive-tutorial/8.png)

3. Microsoft 365 앱이 다운받아질 때까지 잠시만 기다려주세요

![image](/images/m365/onedrive-tutorial/9.png)

- 그리고 나면 Onedrive를 포함한 Microsoft 365 Office 앱 설치가 완료됩니다!

# STEP 4. 앱을 통해 많은 파일 다루기

웹 Onedrive와 달리 앱 Onedrive를 사용했을 때 더 많은 기능을 사용할 수 있습니다.

![image](/images/m365/onedrive-tutorial/10.png)

예를 들어, 아래의 기능들을 더 원활하게 사용할 수 있습니다.

1. 설치형 OneDrive는 운영체재와 직접 통합되기 때문에 훨씬 빠르게 사용 가능합니다.
2. PC의 로컬 폴더와 클라우드가 자동·실시간으로 동기화됩니다.
3. 학교 Wi-Fi가 불안정해도 작업이 가능하고, 수업 중에도 대비할 수 있습니다.
4. 오프라인에서 파일을 편집하면 이후에 자동으로 업로드됩니다.
5. 특정 폴더를 자동 백업하는 기능을 제공합니다.

이를 통해 아래의 업무들을 할 수 있습니다!👍 

- 교안과 수업자료를 빠르게 제작하고 수정, 배포할 수 있어요
    - 수업 직전의 교안 수정이나 빠른 업데이트가 가능해요
- 학생들의 과제를 걷고 관리하는 걸 효율적으로 처리할 수 있어요
- 학교 PC나 집 PC 어디서든 동일한 환경과 작업으로 수업 준비를 할 수 있어요
- USB 실수나 파일 삭제, 실수로 덮어쓰는 사고를 막을 수 있어요
- 대용량 수업 영상이나 학습 자료를 관리할 수 있어요
- 손쉽게 업무 인수인계 문서 공유나 공동 편집을 할 수 있어요
