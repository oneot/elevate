## 1. OneNote ClassNote (수업용 전자필기장)의 구조

원노트는 수업용 전자필기장 기능을 통해 교사의 원노트 아래에 학생들의 원노트를 가져와서 수업에 적합한 형태로 사용할 수 있습니다. 수업용 파일을 즉시 배부할 수도 있고, 학생들의 활동을 실시간으로 확인할 수도 있습니다. 또한 필기를 할 수도 있고 수학문제 등을 풀이해 주는 기능도 있습니다.

## 2. 수업용 전자필기장의 생성

수업용 전자필기장은 되도록 Teams에서 생성하는 것을 권장합니다. 단독으로 수업용 전자필기장을 만들수도 있지만 전입생이 있을 경우 중간에 학생이 추가가 잘 안되는 오류가 있습니다. 그래서 수업용 전자필기장은 Teams를 통해서 생성하는 것을 권장합니다.

### | 팀즈에서 수업용 전자필기장 생성하기

![로컬 이미지.png](/images/onenote/onenote-classnote/imagea.jpg)

1) 팀즈에서 [Class Notebook]을 선택합니다.

2) 기존 전자필기장을 활용할지 빈 전자필기장을 사용할지를 선택합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/imagec.jpg)

3)  학생에게 할당될 섹션을 설정합니다. 섹션이 이름을 바꿀 수도 있고 추가 또는 제거할 수 있습니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/imagee.jpg)

4) 완성된 수업용 전자필기장의 모습입니다.

## 3. 수업용 전자필기장의 기본 활용

수업용 전자 필기장의 [수업용 전자 필기장] 메뉴를 활용하여 수업과 관련된 다양한 활용이 가능합 니다. 그중 가장 기본이 되는 기능이 [배포] 기능입니다. [수업용 전자 필기장]-[페이지 배포]를 통해 다양한 배포가 가능합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image15.jpg)

1) 페이지 배포 : 전체 학생에게 배포

2) 개별 배포 : 전입생 등 특정 학생에게 배포

3) 그룹 배포 : 그룹별 배포

4) 교차 전자 필기장 배포 : 현재 팀이 아닌 다른 팀으로 배포 (예: 1-1반에서 1-2반으로 배포가 가능)

5) 페이지 삭제 : 배포한 페이지를 회수

### | 페이지 배포하기

![로컬 이미지.png](/images/onenote/onenote-classnote/image1a.jpg)

1) [페이지 배포]를 통해 전체 학생에게 배포합니다. 페이지를 배포할 때는 배포하고자 하는 페이지를 열어 놓고 배포를 합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image1d.jpg)

2) 학생들에게 어떤 섹션으로 보낼지를 결정합니다.

3) 배포를 시작하면 1~2분 내외로 배포가 마무리 됩니다.

4) 만약 배포를 잘못했다면 [페이지 삭제]를 하면 됩니다.

## 4. 수업용 전자필기장의 심화 활용

### | 학생 과제물 검토

학생 과제물 검토 기능은 학생들의 과제물을 검토하기 위해 더는 전자필기장을 사용하지 못하도록 잠그는 기능입니다

![로컬 이미지.png](/images/onenote/onenote-classnote/image21.jpg)

1) [학생 과제물 검토]를 선택합니다.

2) 검토할 섹션을 선택합니다

![로컬 이미지.png](/images/onenote/onenote-classnote/image22.jpg)

3) 검토할 페이지를 선택합니다.

4) [페이지 잠금]을 클릭하여 잠금 할 학생을 선택합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image23.jpg)

5) 체크박스를 선택 후 [적용]을 클릭하면 잠금이 실행됩니다

6) 만약 잠금을 풀고 싶다면 체크박스를 해제 하고 [적용]을 해줍니다.

## 5. 수업용 전자필기장 기타 기능

### | 수학 도우미 기능

수학 도우미 기능은 펜으로 문제를 쓰면 풀이 과정과 답안을 제공해 주는 기능입니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image27.jpg)

1)  [그리기] 기능을 이용하여 수식을 작성합니다. 그리고 올가미를 이용하여 수식을 둘러 선택한 후 [수학 도우미]를 클릭합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image28.jpg)

2) [수학 도우미]를 통해 다음과 같이 해를 구하거나 해를 찾는 단계를 제시해 줍니다

### | 그리기와 자 기능

![로컬 이미지.png](/images/onenote/onenote-classnote/image2b.jpg)

1)  [그리기] 기능을 이용하면 디바이스를 통해 손글씨를 쓰거나 그림을 그리는 등의 다양한 활동이 가능합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image2c.jpg)

2)  [자] 기능으로 정밀한 도형 그리기와 그래프 그리기가 가능합니다

## 6. 수업용 전자필기장을 팀즈 외부 원노트 앱으로 열기

### | 팀즈의 수업용 전자필기장을 OneNote for Windows 10 앱으로 열기

팀즈에서 생성한 수업용 전자필기장은 팀즈내에서 열면 화면을 완전히 크게 펼쳐서 보지 못하는 불편함과 동시에 외부 프린트 등을 원노트로 인쇄하는 기능을 사용할 수 없습니다.

원노트 오피스 버전은 다소 복잡하지만 수업에 적합하도록 간소화 된 OneNote for Windows 10 앱이 있습니다. 이 앱을 활용하면 팀즈 밖에서도 편리하게 수업용 전자필기장을 활용할 수 있습니다.

### | OneNote for Windows 10 앱 다운로드 및 설치하기

1) OneNote for Windows 10 앱은 MS Store를 통해서 설치할 수 있습니다.

    [OneNote for Windows 10 설치하기](https:/apps.microsoft.com/store/detail/onenote/9WZDNCRFHVJL?hl=en-us&gl=us)

2) 설치를 완료하면 MS 365 원노트 계정을 입력합니다.

3) 팀즈에서 생성한 수업용 전자필기장을 OneNote for Windows 10으로 여는 방법입니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image31.jpg)

4) 팀즈의 수업용 전자필기장에서 우즉 상단의 [편집하기]를 선택합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image33.jpg)

5) [데스크톱에서 열기]를 선택합니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image35.jpg)

6) [OneNote for Windows 10] 을 클릭하면 OneNote for Windows 10 에서 열리게 됩니다.

![로컬 이미지.png](/images/onenote/onenote-classnote/image37.jpg)

7) [OneNote for Windows 10] 앱으로 열린 모습입니다.
