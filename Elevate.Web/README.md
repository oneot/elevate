
# Elevate Web

Elevate 공개 블로그 SPA입니다. React + Vite 기반으로 빌드되며, Azure Static Web Apps(`swa-elv-web-test`)에 배포됩니다.

## 빠른 시작

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```
